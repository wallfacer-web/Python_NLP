## 一、什么是词频统计？
词频统计就是：

计算一个词在文本中出现了多少次。

例如一句文本里：

```text
python is useful and python is interesting
```

那么：
- `python` 出现 2 次
- `is` 出现 2 次
- `useful` 出现 1 次

这就是最基础的词频。

## 二、为什么它是最直观的文本分析方法之一？
因为它非常接近人的阅读直觉。

当我们快速翻一本书、看一批评论或读一组新闻时，脑子里会自然形成这样的感觉：
- 哪些词反复出现
- 哪些主题最常被提到
- 这批文本大概在说什么

词频统计就是把这种直觉变成可计算、可展示的数字。

## 三、最重要的边界：词频不等于重要性
这句话必须反复强调。

出现次数高，不等于语义最重要。

例如英文文本里常见高频词：
- `the`
- `and`
- `of`
- `to`

这些词出现很多次，但往往不代表主题。

所以词频统计能告诉我们：
- 哪些词提得多

但不能单独告诉我们：
- 哪些词最关键
- 哪些词最有解释力
- 哪些词最能代表主题

后面我们学主题分析、TF-IDF、分类时，会进一步解决这个问题。

## 四、做词频统计前为什么通常要先清洗文本？
如果不先清洗，词频结果通常会很乱。

常见问题包括：
- `Python` 和 `python` 被算成两个词
- `hello` 和 `hello,` 被算成两个词
- 空白、换行、符号干扰结果
- 重复噪声把统计带偏

所以常见顺序是：
1. 先清洗文本
2. 再分词
3. 再统计词频
4. 再做可视化

## 五、为什么要考虑分词和停用词？
### 1. 分词
词频统计的前提是：
- 你得先知道“哪个是词”

英文通常可以先按空格切开。
中文通常要借助分词工具。

### 2. 停用词
停用词指的是：
- 出现很多
- 但对理解主题帮助不大的词

英文里常见停用词：
- `the`
- `is`
- `in`
- `of`

如果不处理停用词，高频词榜单很容易失去解释力。

## 六、最适合初学者的词频统计工具：Counter
Python 里最适合初学者的做法是用 `Counter`。

先导入：

```python
from collections import Counter
```

可以把 `Counter` 理解成：
- 一个自动计票员
- 你把词列表交给它，它会自动数出每个词出现几次

## 七、一个最小词频统计例子

```python
from collections import Counter

clean_text = "apple orange banana apple apple orange grape banana apple"
words = clean_text.split()

word_counts = Counter(words)
print(word_counts)
```

### 代码解释
- `split()`：把字符串切成单词列表
- `Counter(words)`：统计列表中每个词的出现次数

## 八、如何输出 top 10 高频词

```python
from collections import Counter

text = "apple orange banana apple apple orange grape banana apple"
words = text.split()
counts = Counter(words)

top_words = counts.most_common(10)
print(top_words)
```

`most_common(10)` 的意思是：
- 按次数从高到低排序
- 取前 10 个

如果文本不长，可能不足 10 个，也没关系。

## 九、适合课堂的完整小例子

```python
from collections import Counter

text = """
Python is useful. Python is practical.
Natural language processing is interesting.
Python helps us learn NLP.
"""

text = text.lower()
words = text.split()

counts = Counter(words)

print("Top 10 高频词：")
for word, freq in counts.most_common(10):
    print(word, freq)
```

这段代码很适合第一次课堂演示，因为它只依赖：
- 字符串
- `lower()`
- `split()`
- `Counter`

## 十、如果想去掉一些停用词
最简单的入门方式，不一定要先用复杂工具，可以先手工做一个小停用词表：

```python
from collections import Counter

text = "python is useful and python is interesting and python is practical"
words = text.lower().split()

stopwords = {"is", "and"}
filtered_words = []

for word in words:
    if word not in stopwords:
        filtered_words.append(word)

counts = Counter(filtered_words)
print(counts.most_common(10))
```

这个例子很适合课堂，因为学生能马上看出：
- 去停用词前和去停用词后的榜单差别

## 十一、词频可视化的两种基础方式
### 1. 条形图
优点：
- 清楚
- 精确
- 适合展示 top 10 或 top 20 高频词

缺点：
- 展示的词数有限
- 视觉冲击力一般

### 2. 词云
优点：
- 一眼能看出哪些词更突出
- 视觉效果强
- 适合课堂展示和报告展示

缺点：
- 不够精确
- 不能很好比较接近的词频
- 容易让学生误以为“大就是更重要”

## 十二、条形图最小示例

```python
from collections import Counter
import matplotlib.pyplot as plt

text = "python python nlp text text text language language processing"
words = text.split()
counts = Counter(words)

top_words = counts.most_common(5)

labels = [item[0] for item in top_words]
values = [item[1] for item in top_words]

plt.bar(labels, values)
plt.title("Top Words")
plt.xlabel("Word")
plt.ylabel("Frequency")
plt.show()
```

## 十三、词云最小示例
如果本机已经安装 `wordcloud`，可以这样写：

```python
from wordcloud import WordCloud
import matplotlib.pyplot as plt

text = "python python python nlp nlp text analysis language processing"

wc = WordCloud(width=800, height=400, background_color="white")
wc.generate(text)

plt.imshow(wc, interpolation="bilinear")
plt.axis("off")
plt.show()
```

注意：
- 词云更适合展示氛围
- 不适合代替精确统计表

## 十五、初学者最常见错误
### 1. 忘记先分词
如果直接把字符串交给 `Counter`，它会按字符统计，而不是按词统计。

### 2. 忘记统一大小写
导致：
- `Python`
- `python`

被算成两个词。

### 3. 没去标点
导致：
- `hello`
- `hello,`

被算成两个不同词。

### 4. 直接把停用词榜单当结论
如果高频词全是功能词，那说明你还没有做足够的预处理。

### 5. 只看图，不解释
图表只是结果展示，不是分析本身。

学生必须学会回答：
- 为什么这些词高频
- 这些词和文本主题有什么关系
- 结果有哪些局限
