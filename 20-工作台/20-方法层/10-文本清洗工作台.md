# 文本清洗工作台

## 工作台定位
方法层。负责把已经读入 Python 的文本做规范化处理，让它更适合后续的抽取、统计、分类和建模。

这一页不是讲“高级模型”，而是讲最容易被忽视、但最影响结果的前置动作：
- 文本里哪些东西算噪声
- 哪些清洗动作值得先做
- 哪些内容不能乱删

## 一、文本清洗是什么？
文本清洗就是把原始、杂乱、不统一的文本，处理成更干净、更稳定、更适合计算机继续分析的文本。

可以把它理解成：

不是改写文本内容，而是先把文本表面上的混乱整理好。

例如：
- 多余空格
- 多余换行
- 空行
- 大小写不统一
- 明显重复内容
- 一些与当前任务无关的简单符号

## 二、为什么文本清洗不能跳过？
因为计算机不会自动忽略噪声。

如果你直接把原始文本拿去做词频统计、关键词提取或分类，可能会出现这些问题：
- 大量空格和换行干扰结果
- `Python` 和 `python` 被算成两个不同词
- 空白文本让程序报错
- 重复文本让统计失真
- 本来没意义的符号被当成高频特征

一句话：

垃圾输入，通常只会得到混乱输出。

## 三、什么是噪声文本？
噪声不是“看起来奇怪的内容”，而是对当前任务没有帮助、甚至会干扰分析的内容。

常见噪声包括：
- 网页残留标签
- 多余空格
- 多余换行
- 空行
- 纯空白文件
- 明显重复文本
- 与任务无关的部分简单符号

要注意：

噪声不是固定不变的。

比如：
- 做情感分析时，感叹号可能很重要
- 做信息抽取时，数字、日期、邮箱可能很重要

所以清洗不是“删得越多越好”，而是“为任务服务”。

## 四、最基础的清洗动作
下面这些动作，是初学者最应该先掌握的。

### 1. 统一大小写

```python
text = text.lower()
```

作用：
- 把 `Apple` 和 `apple` 统一起来

适合：
- 英文文本

不适用于：
- 中文文本没有大小写，这一步通常不需要

### 2. 去掉首尾空白

```python
text = text.strip()
```

作用：
- 删除字符串开头和结尾的空格、换行符、制表符

### 3. 统一空格

```python
text = " ".join(text.split())
```

作用：
- 把多个连续空白压缩成一个空格

这个写法很适合初学者，因为它比正则更直观。

### 4. 统一换行

```python
text = text.replace("\r\n", "\n").replace("\r", "\n")
```

作用：
- 把不同系统下的换行方式整理成统一形式

### 5. 删除空行

```python
lines = text.splitlines()
clean_lines = []

for line in lines:
    if line.strip() != "":
        clean_lines.append(line.strip())

text = "\n".join(clean_lines)
```

### 6. 删除明显重复文本

```python
texts = ["good", "good", "bad", "nice"]
unique_texts = list(set(texts))
print(unique_texts)
```

注意：
- `set()` 会打乱顺序
- 适合短评、标题、评论
- 不适合小说、演讲、访谈全文

### 7. 去掉简单符号

```python
import re

text = re.sub(r"[^a-zA-Z0-9\\s]", "", text)
```

这适合英文入门演示，但不能机械套用到所有任务。

## 五、一个适合课堂的最小清洗函数

```python
import re

def clean_text(raw_text):
    text = raw_text.lower()
    text = text.strip()
    text = re.sub(r"[\r\n]+", " ", text)
    text = re.sub(r" +", " ", text)
    text = re.sub(r"[^a-z0-9\\s]", "", text)
    return text

dirty_text = "   WOW!!! Natural    Language Processing is   soOOO FuN.\n\n"
cleaned = clean_text(dirty_text)

print("清洗前：", dirty_text)
print("清洗后：", cleaned)
```

这一段代码很适合课堂演示，因为它一次串起了：
- 转小写
- 去首尾空白
- 统一换行
- 统一空格
- 去简单符号

## 六、一步一步解释这个函数
### 第一步：`lower()`
把英文统一成小写。

### 第二步：`strip()`
清掉文本两端看不见的空白。

### 第三步：`re.sub(r"[\\r\\n]+", " ", text)`
把一个或多个换行替换成空格。

### 第四步：`re.sub(r" +", " ", text)`
把多个连续空格压成一个空格。

### 第五步：`re.sub(r"[^a-z0-9\\s]", "", text)`
删掉英文字母、数字、空格以外的简单符号。

## 七、中文文本和英文文本清洗有什么不同？
### 英文文本
常见动作：
- 转小写
- 统一空格
- 展开缩写
- 去简单符号

### 中文文本
常见动作：
- 去空白
- 统一换行
- 去无关符号
- 后续接中文分词

最大的区别是：
- 英文单词之间天然有空格
- 中文没有天然空格，后面通常要借助分词工具

所以中文文本清洗不能完全照抄英文规则。

## 八、什么时候不要过度清洗？
这是这页最重要的边界之一。

### 1. 做情感分析时
不要轻易删掉：
- `not`
- `no`
- `!`

因为这些内容可能直接影响情感方向。

例如：

```text
This movie is not good.
```

如果把 `not` 删掉，意思就完全变了。

### 2. 做信息抽取时
不要乱删：
- 数字
- 日期
- 邮箱
- 电话号码

因为这些往往正是你要抽取的目标。

### 3. 做篇章文本时
不要乱去重。

小说、访谈、演讲这些长文本，本来就依赖上下文顺序。直接 `set()` 会把结构打散。

### 例子1：去掉首尾空白


### 例子2：转小写

```python

```

### 例子3：统一空格


### 例子4：删除空行

```python

    if line.strip() != "":
        result.append(line.strip())

print(result)
```

### 例子5：删除明显重复评论

```python
comments = ["good", "good", "bad", "nice", "nice"]
print(list(set(comments)))
```

### 例子6：清洗 DataFrame 里的文本列

```python
import pandas as pd

df = pd.DataFrame({
    "review": ["  I LOVE Python!  ", "NLP is   great.\n", None]
})

df = df.dropna(subset=["review"])
df["review"] = df["review"].str.lower().str.strip()
df["review"] = df["review"].apply(lambda x: " ".join(x.split()))

print(df)
```

## 十一、初学者最常见错误
### 1. 以为字符串会自动被改掉
错误写法：

```python
text.strip()
```

正确写法：

```python
text = text.strip()
```

### 2. 没看原始数据就套清洗规则
很多同学会直接复制一段“去符号代码”，结果把需要保留的信息也删掉了。

### 3. 过度去重
看到重复就删，但忽略了文本顺序的重要性。

### 4. 中文和英文用完全相同的规则
这是很常见的问题。中英文后续处理链不同，清洗方式不能完全照搬。

### 5. 一次做太多动作，自己都解释不清
教学里更好的方式是：
- 一次只加一个清洗动作
- 每加一步就看一次结果
