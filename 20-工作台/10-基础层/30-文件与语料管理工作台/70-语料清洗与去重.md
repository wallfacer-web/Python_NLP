# 语料清洗与去重

## 这页讲什么？
这页讲：
- 为什么文本导入后不能直接分析
- 什么叫“最基础的语料清洗”
- 初学者能立刻上手的清洗动作有哪些
- 什么叫去重，什么时候适合去重，什么时候不适合
- 适合课堂练习的 Python 例子

## 一、为什么语料导入后不能直接分析？
因为原始文本通常不干净。

你收集来的语料可能会带着很多“杂质”：
- 行首和行尾多余空格
- 空行
- 不统一的大小写
- 多余换行
- 重复句子
- 空文件

如果带着这些问题直接做词频统计、关键词提取或情感分析，结果就会很乱。

所以在真正分析之前，要先做一轮基础清洗。

## 二、最基础的五个清洗动作
### 1. 去掉首尾空白

```python
text = text.strip()
```

### 2. 统一大小写

```python
text = text.lower()
```

### 3. 删除空行和空文件

```python
if text != "":
    print("这行有内容")
```

### 4. 统一换行和空白

```python
text = text.replace("\n", " ")
```

或者：

```python
text = " ".join(text.split())
```

### 5. 去重

```python
unique_texts = list(set(texts))
```

但要注意：`set()` 去重会打乱顺序。

## 三、什么时候适合去重，什么时候不适合？
### 适合去重
- 微博评论
- 商品评价
- 新闻标题
- 短句问答

### 不适合直接去重
- 小说
- 长篇访谈
- 演讲全文
- 章节型文章

## 四、一个最基础的清洗示例

```python
raw_corpus = [
    "  Hello NLP World! \n",
    "I LOVE Python. ",
    "  Hello NLP World! \n",
    "\n",
    "   "
]

clean_corpus = []

for line in raw_corpus:
    clean_line = line.strip()
    clean_line = clean_line.lower()

    if clean_line != "":
        clean_corpus.append(clean_line)

print("清洗后但未去重：")
print(clean_corpus)

unique_corpus = list(set(clean_corpus))

print("去重后：")
print(unique_corpus)
```

## 五、一步一步解释上面的代码
### 第一步：准备原始语料
`raw_corpus` 是一个列表，里面每一项代表一行文本。

### 第二步：准备新列表
`clean_corpus = []` 用来装洗干净的结果。

### 第三步：逐行处理
`for line in raw_corpus:` 表示把每一行都拿出来清洗一次。

### 第四步：去首尾空白
`line.strip()` 不会修改原来的字符串，而是返回一个新字符串。

### 第五步：统一小写
`lower()` 也是返回一个新字符串。

### 第六步：过滤空行
如果洗完以后是空字符串，就不放进结果里。

### 第七步：去重
`set()` 可以去掉重复项，但顺序可能会变。

## 六、适合初学者的课堂例子
### 例子1：去掉姓名前后的空格

```python
name = "  Alice  "
clean_name = name.strip()
print(clean_name)
```

### 例子2：统一大小写

```python
word = "PyThOn"
print(word.lower())
```

### 例子3：删除空行

```python
lines = ["hello", "", "python", "   ", "nlp"]

result = []

for line in lines:
    clean_line = line.strip()
    if clean_line != "":
        result.append(clean_line)

print(result)
```

### 例子4：统一空格

```python
text = "I   love    Python   and   NLP."
clean_text = " ".join(text.split())
print(clean_text)
```

### 例子5：简单去重

```python
comments = ["good", "good", "bad", "nice", "nice"]
unique_comments = list(set(comments))
print(unique_comments)
```

### 例子6：删除空文件

```python
from pathlib import Path

for file_path in Path("1_raw_data").glob("*.txt"):
    text = file_path.read_text(encoding="utf-8").strip()
    if text == "":
        print("空文件：", file_path.name)
```

## 八、常见错误
- 写了 `line.strip()`，但没有重新赋值
- 以为 `lower()` 会自动修改原字符串
- 一上来就对长篇小说做 `set()` 去重
- 清洗后的结果直接覆盖原始文件
- 只顾着去重，不保留原始版本

## 九、一个完整的小案例：读文件、清洗、保存

```python
from pathlib import Path

raw_path = Path("1_raw_data/demo.txt")
clean_path = Path("2_clean_data/demo_clean.txt")

text = raw_path.read_text(encoding="utf-8")
text = text.strip()
text = text.lower()
text = " ".join(text.split())

clean_path.parent.mkdir(exist_ok=True)
clean_path.write_text(text, encoding="utf-8")

print("清洗完成")
```
