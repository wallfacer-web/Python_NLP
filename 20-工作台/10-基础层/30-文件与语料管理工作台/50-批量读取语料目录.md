# 批量读取语料目录

## 这页讲什么

这页讲：
- 为什么 NLP 需要批量读取文件夹中的文本
- 什么是目录、路径、文件名
- 如何遍历目录
- 如何筛选特定后缀文件
- 如何把多个 txt 批量读入列表或字典

## 一、为什么 NLP 需要批量读取语料目录

真实的 NLP 场景中，语料通常不是一篇文章，而是很多篇文章。

例如：
- 1000 篇新闻
- 5000 条评论
- 一整套演讲文本
- 一个作者的全部作品

你不可能手工把每个文件名一条条写进代码里。  
所以程序必须学会自己去文件夹里“找文件、读文件、装进语料库”。

## 二、什么是目录、路径、文件名

可以用图书馆来类比：

- 目录 / 文件夹：像书架
- 文件名：像书名
- 路径：像去书架的路线

例如：

```text
data/article1.txt
```

这里：
- `data` 是文件夹
- `article1.txt` 是文件名

## 三、最常见的三个工具

### 1. `os.listdir()`

只列出当前文件夹这一层的内容。

```python
import os

names = os.listdir("data")
print(names)
```

### 2. `os.walk()`

不仅看当前文件夹，还会继续看子文件夹。

```python
import os

for foldername, subfolders, filenames in os.walk("data"):
    print(foldername)
    print(filenames)
```

### 3. `pathlib`

这是更现代、更清楚的方式。

```python
from pathlib import Path

folder = Path("data")
for file_path in folder.glob("*.txt"):
    print(file_path)
```

## 四、如何筛选特定后缀文件

### 用 `endswith()`

```python
import os

for filename in os.listdir("data"):
    if filename.endswith(".txt"):
        print(filename)
```

### 用 `glob("*.txt")`

```python
from pathlib import Path

folder = Path("data")
for file_path in folder.glob("*.txt"):
    print(file_path)
```

## 五、如何批量读取 txt 并放入列表

这是最常见的 NLP 起点动作。

```python
import os

corpus = []

for filename in os.listdir("data"):
    if filename.endswith(".txt"):
        path = os.path.join("data", filename)
        with open(path, "r", encoding="utf-8") as f:
            corpus.append(f.read())

print(f"成功读取 {len(corpus)} 篇文本")
```

这样得到的 `corpus`，就是一个“文本列表”。

## 六、如何同时记录文件名和文本内容

有时候不能只保存文本本身，还需要知道：
- 这篇文本来自哪个文件

这时可以用字典。

```python
import os

corpus_dict = {}

for filename in os.listdir("data"):
    if filename.endswith(".txt"):
        path = os.path.join("data", filename)
        with open(path, "r", encoding="utf-8") as f:
            corpus_dict[filename] = f.read()

print(corpus_dict)
```

这里：
- 键是文件名
- 值是文本内容

## 七、适合课堂演示的例子

### 例子1：列出文件夹中的全部内容

```python
import os

print(os.listdir("data"))
```

### 例子2：只找 txt 文件

```python
import os

for filename in os.listdir("data"):
    if filename.endswith(".txt"):
        print(filename)
```

### 例子3：拼接完整路径

```python
import os

for filename in os.listdir("data"):
    if filename.endswith(".txt"):
        full_path = os.path.join("data", filename)
        print(full_path)
```

### 例子4：用 pathlib 查找 txt

```python
from pathlib import Path

folder = Path("data")
for file_path in folder.glob("*.txt"):
    print(file_path)
```

### 例子5：批量读取到列表

```python
from pathlib import Path

folder = Path("data")
corpus = []

for file_path in folder.glob("*.txt"):
    text = file_path.read_text(encoding="utf-8")
    corpus.append(text)

print(corpus)
```

### 例子6：同时保存文件名和文本

```python
from pathlib import Path

folder = Path("data")
corpus = []

for file_path in folder.glob("*.txt"):
    text = file_path.read_text(encoding="utf-8")
    corpus.append({"title": file_path.name, "content": text})

print(corpus)
```

### 例子7：判断文本里有没有关键词

```python
from pathlib import Path

folder = Path("data")

for file_path in folder.glob("*.txt"):
    text = file_path.read_text(encoding="utf-8")
    if "Python" in text:
        print("找到关键词：", file_path.name)
```

### 例子8：遇到坏文件时跳过

```python
from pathlib import Path

corpus = []

for file_path in Path("data").glob("*.txt"):
    try:
        text = file_path.read_text(encoding="utf-8")
        corpus.append(text)
    except Exception as e:
        print(f"跳过坏文件 {file_path.name}，原因：{e}")
```

## 九、常见错误

- 只拿到了文件名，没有拼接完整路径
- 路径写错，报 `FileNotFoundError`
- 忘记写 `encoding="utf-8"`
- 把文件夹和文件名搞混
- 一次性读太多文件导致内存压力过大
