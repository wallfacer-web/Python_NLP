# 模块的使用与调用

## 这页讲什么

这页讲 Python 里的模块。  
目标不是让学生立刻掌握复杂的工程项目结构，而是让学生知道：为什么很多功能不用自己从零写，以及后面的自然语言处理为什么一定会大量依赖模块。

## 一、什么是模块

最简单的理解：

模块就是一个 `.py` 文件。  
这个文件里面可以放：
- 函数
- 类
- 变量
- 已经写好的工具代码

你可以把模块想象成一个“专业工具箱”。

例如：
- 一个工具箱专门处理数学问题
- 一个工具箱专门处理文件路径
- 一个工具箱专门处理文本

你不需要自己造每一把工具，而是学会把工具箱拿来使用。

## 二、为什么要用模块

### 1. 避免重复造轮子

很多功能前人已经写好了，比如：
- 数学计算
- 随机抽样
- 文件路径处理
- 正则表达式
- 文本分析

如果每次都自己重写，效率太低。

### 2. 让代码更清楚

如果把所有代码都塞进一个文件里，代码会很乱。  
把不同功能拆到不同模块里，会更有条理。

### 3. 后续 NLP 一定会依赖模块

自然语言处理不是只靠几行基础代码就能完成的。  
后面我们会接触：
- `re`
- `pathlib`
- `pandas`
- `jieba`
- `nltk`
- `spacy`

这些都是模块或库。  
所以，学会“导入模块”和“调用模块中的工具”，是进入 NLP 的门槛之一。

## 三、模块、函数、类之间是什么关系

这一部分一定要理清。

你可以这样理解：

- 函数：像一个专门干活的小助手
- 类：像一张设计图纸
- 模块：像一个办公室或工具箱，里面装着很多函数和类

一句话总结：

- 模块是容器
- 函数和类是模块里的内容

## 四、最常见的导入方式

## 1. `import 模块名`

这表示：把整个工具箱搬过来。

例如：

```python
import math

print(math.sqrt(9))
```

这里：
- `math` 是模块名
- `sqrt()` 是模块里的一个函数

你必须写成 `math.sqrt(9)`，因为 Python 需要知道这个工具来自哪个模块。

## 2. `from 模块名 import 工具名`

这表示：不搬整个工具箱，只把某个工具拿出来。

例如：

```python
from math import sqrt

print(sqrt(9))
```

这里就不需要再写 `math.` 了。

## 3. `as` 起别名

有些模块名字比较长，或者你想少打字，可以起别名。

例如：

```python
import pathlib as pl

file_path = pl.Path("story.txt")
print(file_path)
```

这里：
- `pathlib` 是原名
- `pl` 是别名

## 五、为什么要学会区分三种写法

### `import 模块名`

优点：
- 更清楚
- 代码一看就知道工具来自哪里

适合初学者先用。

### `from 模块名 import 工具名`

优点：
- 写起来更短

但如果导入太多同名函数，容易混乱。

### `as`

优点：
- 简洁
- 适合常用模块

例如：

```python
import pandas as pd
```

后面你会经常看到这种写法。

## 六、模块为什么对自然语言处理很重要

因为 NLP 的很多能力本身都很复杂。

例如：
- 文本切词
- 词性标注
- 命名实体识别
- 文本分类
- 情感分析

这些功能背后通常有大量算法和规则。  
作为初学者，我们不需要从零手写所有底层逻辑，而是先学会：

1. 导入模块
2. 调用模块
3. 看懂输入和输出

这也是文科学生进入 Python 与 NLP 最现实的路径。

## 七、最基础的模块例子

### 例子1：导入数学模块

```python
import math

print(math.sqrt(16))
```

### 例子2：只导入一个函数

```python
from math import sqrt

print(sqrt(25))
```

### 例子3：随机抽一个词

```python
from random import choice

words = ["love", "peace", "joy", "empathy"]
today_word = choice(words)
print("今日主题词是：", today_word)
```

这个例子适合文本处理入门，因为它已经开始在“词语列表”上操作了。

### 例子4：用 pathlib 读取文件路径

```python
import pathlib as pl

file_path = pl.Path("story.txt")
print(file_path)
```

### 例子5：读取文本内容

```python
import pathlib as pl

file_path = pl.Path("story.txt")
content = file_path.read_text(encoding="utf-8")

print(content)
```

### 例子6：用 re 模块清理数字

```python
import re

messy_text = "我今天吃了3个苹果，走了5000步。"
clean_text = re.sub(r"\d+", "几", messy_text)

print(clean_text)
```

输出会接近：

```text
我今天吃了几个苹果，走了几步。
```

这已经非常接近真实文本清洗任务。

## 八、和文本处理有关的初学者练习

### 练习1：导入 `math` 计算平方根

```python
import math
print(math.sqrt(36))
```

### 练习2：用 `random` 随机选一个词

```python
from random import choice

words = ["文学", "语言", "文本", "修辞"]
print(choice(words))
```

### 练习3：给模块起别名

```python
import pathlib as pl

path_obj = pl.Path("note.txt")
print(path_obj)
```

### 练习4：读取一个文本文件

```python
import pathlib as pl

file_path = pl.Path("article.txt")
content = file_path.read_text(encoding="utf-8")
print(content)
```

### 练习5：统计文本里有没有某个词

```python
import pathlib as pl

file_path = pl.Path("article.txt")
content = file_path.read_text(encoding="utf-8")

print("Python" in content)
```

### 练习6：用 `re` 替换文本中的年份

```python
import re

text = "2024年和2025年都是重要年份。"
new_text = re.sub(r"\d{4}", "某年", text)
print(new_text)
```

### 练习7：用 `re` 删除数字

```python
import re

text = "第1章有20页。"
new_text = re.sub(r"\d+", "", text)
print(new_text)
```

### 练习8：读取文件后转成小写

```python
import pathlib as pl

file_path = pl.Path("english.txt")
content = file_path.read_text(encoding="utf-8")
print(content.lower())
```

### 练习9：模块关系判断题

请说出下面每个名字分别是什么：
- `re`
- `sub`
- `Path`

参考理解：
- `re` 是模块
- `sub` 是模块里的函数
- `Path` 是 `pathlib` 模块里的类

## 九、模块导入中的常见错误

- 写了 `import`，但调用时拼错模块名
- 用了 `from ... import ...`，但导入的工具名不存在
- 文件路径不对，导致读取失败
- 没写 `encoding="utf-8"`，中文文本报错
- 把模块、函数、类混为一谈

- 会使用 `import`
- 会使用 `from ... import ...`
- 会使用 `as`
- 能读懂简单的模块调用代码
- 能把模块调用用于最简单的文本处理任务
