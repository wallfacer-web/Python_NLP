# 函数

## 这页讲什么

这页讲 Python 里的函数。  
目标不是让你立刻写很复杂的程序，而是让你理解：为什么代码要“打包”，以及函数怎么帮助你少重复、少出错。

## 一、什么是函数

可以把函数想象成你雇佣的一个“小助手”。

这个小助手有三个特点：
- 它有名字
- 它专门做某一类事情
- 你需要时可以反复叫它出来工作

例如：

```python
def greet_user():
    print("你好，欢迎来到 Python 课堂")
```

这里：
- `def` 表示“我现在要定义一个函数”
- `greet_user` 是函数名
- 缩进部分是这个函数要做的事

## 二、为什么要用函数

如果一段代码你要重复写很多次，直接复制粘贴会很麻烦。

比如你想反复打印欢迎语，如果不用函数，可能会写成：

```python
print("你好，欢迎来到 Python 课堂")
print("你好，欢迎来到 Python 课堂")
print("你好，欢迎来到 Python 课堂")
```

如果以后想改句子，就要每一处都改。

用函数之后：

```python
def greet_user():
    print("你好，欢迎来到 Python 课堂")

greet_user()
greet_user()
greet_user()
```

这样更短，也更清楚。

## 三、函数最重要的价值

- 减少重复代码
- 让代码更像一个个小步骤
- 更方便修改
- 更方便阅读

你可以把函数理解成“把一个动作打包并命名”。

## 四、什么是参数

参数可以理解成你交给小助手的“原材料”。

比如你希望小助手向不同的人打招呼，那你就要把名字交给它。

```python
def greet_user(username):
    print("你好，" + username)

greet_user("张三")
greet_user("李四")
```

这里：
- `username` 是参数
- `"张三"`、`"李四"` 是传进去的具体内容

参数的意义是：  
同一个函数，不必每次都写死，它可以处理不同输入。

## 五、什么是返回值

有时候，函数不只是打印一句话，而是把处理结果“交回来”。

这就叫返回值。

```python
def get_full_name(first_name, last_name):
    return first_name + last_name

name = get_full_name("张", "华")
print(name)
```

这里：
- `return` 的意思是“把结果返回”
- 函数算完以后，把结果交给 `name`

## 六、函数命名怎么做

函数命名要尽量清楚，像一个动作。

推荐：

```python
greet_user
read_text
save_result
count_words
```

不推荐：

```python
aa
doit
x1
```

好的函数名应该让人一眼就知道它要干什么。

## 七、最简单的函数例子

### 例子1：打印欢迎语

```python
def say_hello():
    print("你好，欢迎学习 Python")

say_hello()
```

### 例子2：带参数的问候

```python
def greet_user(username):
    print("你好，" + username + "！")

greet_user("王芳")
```

### 例子3：返回字符串长度

```python
def get_length(text):
    return len(text)

result = get_length("自然语言处理")
print(result)
```

### 例子4：替换文本中的词

```python
def replace_word(text, old_word, new_word):
    return text.replace(old_word, new_word)

result = replace_word("我喜欢北京", "北京", "上海")
print(result)
```

## 八、适合初学者的函数练习

### 练习1：你最喜欢的书

写一个函数 `favorite_book(title)`，打印：

```text
我最喜欢的书之一是《......》
```

参考：

```python
def favorite_book(title):
    print("我最喜欢的书之一是《" + title + "》")

favorite_book("红楼梦")
```

### 练习2：欢迎某位同学

写一个函数，输入学生姓名，输出欢迎语。

### 练习3：统计一句话长度

写一个函数，输入一句话，返回它的长度。

### 练习4：清理首尾空格

写一个函数，输入一段带空格的文本，返回清理后的结果。

参考：

```python
def clean_text(text):
    return text.strip()

print(clean_text("   张三   "))
```

### 练习5：保存一句话到文件

写一个函数，把输入的文本保存到 `note.txt`。

```python
def save_text(text):
    with open("note.txt", "w", encoding="utf-8") as f:
        f.write(text)

save_text("这是我写入文件的第一句话。")
```

## 九、初学者常见错误

- 定义了函数，但忘了调用
- 函数名和调用名拼写不一致
- 少了冒号 `:`
- 缩进不对
- 该 `return` 的地方只写了 `print`

## 十、这一页学完后你应会什么

- 知道函数是“带名字的小助手”
- 知道函数为什么能减少重复
- 知道什么是参数
- 知道什么是返回值
- 能写最简单的函数
