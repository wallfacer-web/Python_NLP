# 正则表达式入门

## 这页讲什么

这页讲正则表达式最基础的用法。  
目标不是让学生一次学会所有规则，而是先学会：如何用 Python 在文本里找模式、替换模式。

## 一、什么是正则表达式

正则表达式可以理解成“找文本规律的工具”。

例如你想找：
- 所有数字
- 所有邮箱
- 所有日期
- 所有连续出现的某种格式

这时候，逐个字硬写会很麻烦，正则表达式就能帮你按模式匹配。

## 二、为什么后面 NLP 会用到它

因为真实文本常常很乱。  
你经常需要先做这些事：
- 删除数字
- 统一符号
- 提取年份
- 找出邮箱、电话、网址

这些都属于文本清洗和规则抽取的基础动作。

## 三、Python 里怎么用正则表达式

Python 里最常用的是 `re` 模块。

先导入：

```python
import re
```

## 四、最常见的基础写法

### 1. `re.findall()`

作用：找到所有符合规则的内容。

```python
import re

text = "我今天读了3页书，写了2段笔记。"
numbers = re.findall(r"\d+", text)
print(numbers)
```

输出：

```python
['3', '2']
```

### 2. `re.sub()`

作用：把符合规则的内容替换掉。

```python
import re

text = "我今天读了3页书，写了2段笔记。"
new_text = re.sub(r"\d+", "几", text)
print(new_text)
```

### 3. `re.search()`

作用：查找文本里有没有某种模式。

```python
import re

text = "课程开始时间是2026年3月。"
result = re.search(r"\d{4}", text)
print(result)
```

如果找到了，会返回一个匹配对象；找不到会返回 `None`。

## 五、最常用的几个基础符号

### `\d`

表示一个数字。

```python
r"\d"
```

### `\d+`

表示一个或多个数字。

```python
r"\d+"
```

### `\w`

表示字母、数字或下划线中的一个字符。

### `\s`

表示空白字符，比如空格、换行。

### `.`

表示任意一个字符。

### `*`

表示前面的模式可以重复 0 次或多次。

### `+`

表示前面的模式可以重复 1 次或多次。

## 六、最适合初学者的例子

### 例子1：找出句子里的数字

```python
import re

text = "2026年我读了12本书。"
print(re.findall(r"\d+", text))
```

### 例子2：把数字删掉

```python
import re

text = "第3章有20页。"
new_text = re.sub(r"\d+", "", text)
print(new_text)
```

### 例子3：把年份替换成“某年”

```python
import re

text = "2024年和2025年都很重要。"
new_text = re.sub(r"\d{4}", "某年", text)
print(new_text)
```

### 例子4：找出邮箱

```python
import re

text = "请联系 test123@example.com 获取资料。"
emails = re.findall(r"\w+@\w+\.\w+", text)
print(emails)
```

### 例子5：统一多余空格

```python
import re

text = "Python    很   有趣"
new_text = re.sub(r"\s+", " ", text)
print(new_text)
```

## 七、初学者练习

### 练习1：提取文本里的所有数字

```python
import re

text = "我买了5本书和12支笔。"
print(re.findall(r"\d+", text))
```

### 练习2：删除一句话中的数字

```python
import re

text = "今天是第2周。"
print(re.sub(r"\d+", "", text))
```

### 练习3：把多个空格替换成一个空格

```python
import re

text = "我   爱   自然语言处理"
print(re.sub(r"\s+", " ", text))
```

### 练习4：提取邮箱

自己写一句带邮箱的文本，再尝试提取。

### 练习5：提取四位年份

```python
import re

text = "2023年、2024年、2025年都出现了。"
print(re.findall(r"\d{4}", text))
```

## 八、常见错误

- 忘记先 `import re`
- 把正则模式写错
- 以为 `\d+` 只能找一个数字
- 看不懂返回结果是列表还是字符串

## 九、这一页学完后你应会什么

- 知道正则表达式是“找文本规律的工具”
- 会使用 `re.findall()`
- 会使用 `re.sub()`
- 能做最简单的数字、空格、年份处理
