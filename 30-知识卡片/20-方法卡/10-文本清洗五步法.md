# 文本清洗五步法

## 方法定位
这张方法卡把“文本清洗工作台”里的核心动作压缩成 5 步，适合零基础学生直接照着做。

## 适用场景
- 课程反馈清洗
- 评论文本预处理
- 规则抽取前的文本整理
- 词频统计前的文本规范化

## 不适用场景
- 需要保留大量原始格式信息的任务
- 需要保留情绪符号、标点强度的情感分析任务
- 需要保留日期、编号、邮箱等字段的抽取任务

## 五步流程

### 第一步：确认任务目标
先回答一个问题：这次清洗是为了什么？

不同任务，清洗规则不同：
- 为分类做准备：通常要统一格式、处理空值、去明显噪声
- 为信息抽取做准备：不能乱删数字、日期、邮箱
- 为情感分析做准备：不能轻易删掉否定词和感叹号

## 第二步：统一基本格式
做最基础的规范化：
- 去掉首尾空白
- 统一换行
- 压缩多余空格
- 英文文本统一大小写

最小例子：

```python
text = text.strip()
text = " ".join(text.split())
text = text.lower()
```

## 第三步：处理明显噪声
清理那些对当前任务没有帮助、反而会干扰统计或模型的内容：
- 空行
- 空文本
- 明显重复文本
- 无意义网页残留字符

最小例子：

```python
texts = [t for t in texts if t.strip() != ""]
texts = list(dict.fromkeys(texts))
```

## 第四步：保留关键信息
清洗不是“删得越多越好”，而是“删噪声、留信息”。

做这一步时要特别检查：
- 日期要不要留
- 数字要不要留
- 标点要不要留
- 大小写差异有没有意义

## 第五步：抽样检查结果
清洗后不要立刻进入下一步，先抽样看几条：
- 清洗前后是不是能对上
- 有没有误删重要信息
- 有没有仍然残留明显噪声

最小例子：

```python
for item in texts[:5]:
    print(item)
```

## 常见错误
- 一开始就套复杂正则，自己也解释不清
- 不看任务目标，直接全删符号和数字
- 用 `set()` 去重却忽略了顺序问题
- 清洗后不抽样检查
