# 文本分类实验流程

## 方法定位
这张方法卡把最基础的文本分类实验拆成稳定步骤，适合第一次做机器学习文本分类的学生。

## 适用场景
- 课程反馈分类
- 评论正负面分类
- 新闻主题分类
- 垃圾邮件识别

## 不适用场景
- 数据极少且类别定义不清
- 完全没有标签的数据
- 需要复杂深层语义推理的超复杂任务

## 六步流程

### 第一步：明确标签
先说清楚这次要分成什么类。

例如：
- 正面 / 负面 / 中性
- 体育 / 财经 / 科技
- 建议 / 表扬 / 抱怨

标签不清楚，后面所有步骤都会乱。

### 第二步：准备数据
确保每条文本都有对应标签，并检查：
- 是否有空值
- 类别是否极不平衡
- 文本是否明显脏乱

### 第三步：做基础清洗
最基本的清洗包括：
- 去空值
- 去多余空白
- 必要时统一大小写

这一步不要过度清洗，先保持稳。

### 第四步：把文本转成特征
初学者最适合先用：
- Bag of Words
- TF-IDF

最小例子：

```python
from sklearn.feature_extraction.text import TfidfVectorizer

vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(texts)
```

### 第五步：训练基础模型
先从简单模型开始，如：
- 朴素贝叶斯
- 逻辑回归

最小例子：

```python
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB

X_train, X_test, y_train, y_test = train_test_split(X, labels, test_size=0.2, random_state=42)
model = MultinomialNB()
model.fit(X_train, y_train)
```

### 第六步：评估与解释
至少看：
- accuracy
- precision
- recall
- F1

还要抽样看错误案例，不能只看一个分数。

## 常见错误
- 标签定义模糊
- 数据没清洗就直接建模
- 只看准确率，不看错误样本
- 模型一跑通就以为任务完成
