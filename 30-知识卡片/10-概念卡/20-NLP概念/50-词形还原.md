# 词形还原

## 一句话定义
词形还原是把单词转换为更接近字典基础形态的过程。

## 通俗理解
词形还原像是把词语“还原回原型”。

例如：
- running -> run
- books -> book

## 最小例子
如果一篇文本里同时出现：
- run
- runs
- running

词形还原可以帮助把它们看作更接近同一个核心词。

## 常见误区
- 把词形还原和词干提取完全混为一谈
- 以为所有场景都必须做词形还原
- 不理解词形还原常常比简单砍词尾更自然
