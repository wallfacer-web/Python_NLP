# 分词

## 一句话定义
分词是把连续文本切分成一个个词或 token 的过程。

## 通俗理解
计算机看到的一大段文字，最开始常常只是连续字符。

分词的作用就是先回答一个最基础的问题：
- 哪些部分应该被当作“一个词”来看

## 最小例子
英文句子：

```text
Python is useful for NLP.
```

分词后可能得到：

```text
["Python", "is", "useful", "for", "NLP", "."]
```

## 常见误区
- 以为英文分词永远等于按空格切开
- 以为中文分词和英文完全一样
- 以为分词只是技术细节，实际上它会直接影响后面的统计和建模
