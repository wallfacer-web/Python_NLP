# 路径

## 一句话定义
路径是文件或文件夹在计算机中的位置表示方式。

## 通俗理解
路径就像地址。

如果程序不知道文件住在哪里，就没法把它读进来。

## 最小例子

```python
from pathlib import Path

file_path = Path("data/demo.txt")
print(file_path)
```

## 常见误区
- 文件明明存在，但路径写错了
- 相对路径和绝对路径分不清
- Windows 路径里的反斜杠容易写乱
