# 解释器

## 一句话定义
解释器是负责读取并执行 Python 代码的程序。

## 通俗理解
可以把解释器想成一个会看 Python 语言说明书的翻译员。

你写下来的代码，本身不会自动运行。解释器会把代码一行一行读进去，然后告诉计算机该做什么。

## 最小例子

```python
print("Hello, Python")
```

当你运行这行代码时，真正执行它的不是文本本身，而是 Python 解释器。

## 常见误区
- 以为 `.py` 文件双击一下，代码就“天然会跑”
- 以为 VS Code 在运行代码，其实真正执行代码的是所选 Python 解释器
- 分不清“编辑器”和“解释器”
