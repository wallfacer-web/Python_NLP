# 类与对象

## 一句话定义
类是对象的设计图，对象是按设计图创建出来的具体实例。

## 通俗理解
类像“模具”，对象像“做出来的成品”。

在课程里，很多 NLP 工具并不是简单返回字符串，而是返回对象，比如 spaCy 的 `Doc`、`Token`。

## 最小例子

```python
import spacy

nlp = spacy.load("en_core_web_sm")
doc = nlp("Python is useful.")
print(type(doc))
```

这里 `doc` 就是一个对象。

## 常见误区
- 一看到类和对象就以为太高级，其实用 NLP 库时早就已经在用对象
- 把对象当成普通字符串使用
- 不理解对象上有属性和方法
