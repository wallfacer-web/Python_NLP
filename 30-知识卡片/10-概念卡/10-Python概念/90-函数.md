# 函数

## 一句话定义
函数是可重复调用的一段命名代码，用来封装某个明确动作。

## 通俗理解
可以把函数想成一个小工具或小助手。

你把输入交给它，它帮你完成动作，然后返回结果。

## 最小例子

```python
def clean_text(text):
    return text.strip().lower()

print(clean_text("  Python NLP  "))
```

## 常见误区
- 只会复制粘贴代码，不会封装成函数
- 分不清“定义函数”和“调用函数”
- 以为函数越长越好，其实一个函数最好只做一件事
