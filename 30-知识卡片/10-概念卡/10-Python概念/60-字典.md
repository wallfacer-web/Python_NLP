# 字典

## 一句话定义
字典是按“键-值对”方式存储数据的结构。

## 通俗理解
字典不像列表靠位置取值，而是靠“名字”取值。

例如：
- `title` 对应标题
- `text` 对应正文
- `label` 对应标签

这非常适合处理结构化文本数据。

## 最小例子

```python
item = {
    "title": "课程反馈",
    "text": "这门课内容很清楚",
    "label": "positive"
}

print(item["text"])
```

## 常见误区
- 把字典和列表混用
- 不理解“键必须先存在”
- 键名设计混乱，后面很难维护
