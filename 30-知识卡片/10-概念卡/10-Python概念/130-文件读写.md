# 文件读写

## 一句话定义
文件读写是用代码打开、读取和保存文件内容的基本操作。

## 通俗理解
你可以把它理解成：
- 把文件里的内容拿进程序
- 把程序处理后的结果再写回文件

## 最小例子

```python
from pathlib import Path

text = Path("demo.txt").read_text(encoding="utf-8")
Path("output.txt").write_text(text.lower(), encoding="utf-8")
```

## 常见误区
- 只会读，不会写
- 读取时忘记编码
- 把原始文件直接覆盖，导致原始数据丢失
