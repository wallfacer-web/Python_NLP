# 模块

## 一句话定义
模块是别人或自己写好的代码文件，可以通过 `import` 导入后直接使用。

## 通俗理解
模块像一个工具箱。

你不需要每次都从零写正则、表格处理、NLP 解析器，而是把现成工具箱导进来用。

## 最小例子

```python
import re

text = "Email: test@example.com"
print(re.findall(r"[a-zA-Z0-9._%+-]+@[a-zA-Z0-9.-]+\.[a-zA-Z]+", text))
```

## 常见误区
- 以为 `import` 很神秘，其实就是“把工具拿进来”
- 不知道 `import pandas as pd` 里的 `pd` 是简写名
- 把模块名和函数名混淆
