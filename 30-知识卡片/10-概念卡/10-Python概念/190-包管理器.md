# 包管理器

## 一句话定义
包管理器是安装和更新 Python 外部库的工具，最常见的是 `pip`。

## 通俗理解
包管理器像 Python 世界里的应用商店。

你想用：
- `pandas`
- `spacy`
- `whisper`
- `edge-tts`

都需要先通过包管理器装进去。

## 最小例子

```bash
pip install pandas
```

## 常见误区
- 只会复制命令，不知道它在干什么
- 装在错误环境里，结果代码里还是导入失败
- 分不清“安装库”和“导入库”
