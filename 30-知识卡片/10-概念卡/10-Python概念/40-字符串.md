# 字符串

## 一句话定义
字符串是用引号括起来的文本内容，是自然语言处理里最核心的数据类型之一。

## 通俗理解
只要是“文字”，在 Python 里大多数时候都会先以字符串的形式出现。

无论是一句话、一个单词、一篇文章、一个文件路径，课程里很多对象最开始都是字符串。

## 最小例子

```python
text = "Natural Language Processing"
print(text.lower())
```

这里 `text` 就是字符串变量。

## 常见误区
- 忘记给文本加引号
- 分不清字符串和数字，比如 `"123"` 和 `123`
- 以为字符串是这门课里“很基础的小概念”，实际上后面几乎所有文本操作都围绕它展开
