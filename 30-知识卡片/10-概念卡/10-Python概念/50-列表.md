# 列表

## 一句话定义
列表是按顺序存放多个元素的数据容器。

## 通俗理解
可以把列表理解成一个有顺序的收纳架。

里面可以放：
- 多个文件名
- 多条评论
- 多篇文章
- 多个结果

## 最小例子

```python
texts = ["I like Python.", "NLP is useful.", "This class is interesting."]
print(texts[0])
```

## 常见误区
- 以为列表里只能放一种东西
- 分不清列表和字符串
- 忘记列表是有顺序的，可以按位置取值
