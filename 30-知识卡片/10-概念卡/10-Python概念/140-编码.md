# 编码

## 一句话定义
编码是文本在计算机中保存和读取时所遵循的字符表示规则。

## 通俗理解
编码像一套“文字翻译规则”。

如果保存文件和读取文件时用的规则不一致，就容易出现乱码。

## 最小例子

```python
text = open("demo.txt", "r", encoding="utf-8").read()
```

## 常见误区
- 以为乱码是“文件坏了”，其实很多时候只是编码不一致
- 不知道 `utf-8` 和 `gbk` 的区别
- 保存和读取时使用了不同编码
