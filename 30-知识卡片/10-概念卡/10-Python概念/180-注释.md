# 注释

## 一句话定义
注释是写给人看的说明文字，运行代码时会被解释器忽略。

## 通俗理解
注释不是给机器看的，而是给自己、同学和老师看的。

它能帮助别人快速知道：
- 这一段代码在做什么
- 为什么这样写

## 最小例子

```python
# 把文本统一转成小写
text = text.lower()
```

## 常见误区
- 要么完全不写注释
- 要么把每一行都写得过于啰嗦
- 以为注释写中文或英文有本质区别，其实关键是清楚
