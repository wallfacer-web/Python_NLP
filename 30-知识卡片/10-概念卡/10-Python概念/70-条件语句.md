# 条件语句

## 一句话定义
条件语句根据条件是否成立，决定程序走哪条分支。

## 通俗理解
可以把条件语句想成程序里的岔路口。

如果条件成立，就走这边；不成立，就走另一边。

## 最小例子

```python
text = "python"

if text != "":
    print("这行不是空文本")
```

## 常见误区
- 条件后面忘记写冒号
- 不理解缩进为什么重要
- 以为条件语句只能用来做数学判断，其实文本过滤也大量依赖它
