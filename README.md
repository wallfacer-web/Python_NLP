# Python 与自然语言处理课程库

这是一个以实践为主的课程系统，不按教材章节组织，而按“使用 Python 解决文本处理问题”的工作流组织。

它不是单纯的知识笔记库，而是由以下几部分组成：

- 知识卡片：解释概念、方法和边界
- 工作台：把知识变成步骤
- 模板：把步骤变成标准输出
- 对象库：提供案例、FAQ、活动和教学对象
- 课程运行：提供周次、作业、评分和项目安排

## 仓库结构

- `10-角色入口`：教师和学生入口
- `20-工作台`：课程执行主线
- `30-知识卡片`：概念卡、方法卡、边界卡
- `40-模板`：作业、实验、项目模板
- `50-对象库`：案例、FAQ、画像、活动
- `60-课程运行`：周次安排、作业说明、评分标准、项目任务

## 建库原则

- 先结构，后内容
- 先工作流，后扩展页面
- 工作台和模板必须成对出现
- 公开站首页不能只是 README
- 公开页面优先使用 wikilink 或相对路径，不保留本地绝对路径

## 本地预览

当前仓库已经接入 Quartz v4 骨架，并通过导出脚本把课程公开内容同步到 `content/`。

常用命令：

```bash
npm install
npm run site:dev
```

如果只想生成静态站点：

```bash
npm run site:build
```
