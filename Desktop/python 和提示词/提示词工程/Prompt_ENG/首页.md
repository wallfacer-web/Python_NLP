# 提示词工程课程首页

本课程不把提示词工程理解为“套模板”，而把它理解为一门围绕大模型协作的系统能力课。

## 你会学到什么

- 把模糊需求改写成结构化指令
- 把知识库接入模型，做上下文工程
- 把单轮对话升级为多步工作流
- 用评测和版本控制迭代提示词
- 设计带工具调用的智能体
- 为输出建立风险边界与护栏
