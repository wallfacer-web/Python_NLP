# 课程运行总览

课程运行层负责把这套内容真正变成一门能开的课。
它关注的是周次、作业、评分和项目，而不是概念解释。

## 主要页面

- [[60-课程运行/周次安排|周次安排]]
- [[60-课程运行/作业说明|作业说明]]

- [[60-课程运行/项目任务|项目任务]]
- [[60-课程运行/教学检查清单|教学检查清单]]

## 它解决什么问题

- 每周上什么
- 每次交什么
- 怎么评分
- 项目怎么推进
- 教师每周怎么检查

## 建议使用顺序

1. 先看周次安排
2. 再看作业说明和评分标准
3. 中后期进入项目任务
4. 教师全程配合教学检查清单
