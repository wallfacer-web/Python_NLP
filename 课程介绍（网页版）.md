# 课程介绍

《Python 与自然语言处理》是一门面向实践的课程。
它不把重点放在抽象定义堆积上，而是围绕一个更直接的问题展开：

如何使用 Python，把原始文本转化为可分析、可解释、可交付的结果？

## 这门课适合谁

- 希望学会 Python 文本处理的初学者
- 需要处理评论、访谈、新闻、课程反馈等文本材料的学生
- 想从传统 NLP 过渡到大模型应用的学习者

## 这门课怎么组织

课程不是按教材章节组织，而是按任务链组织：

1. 搭建基础能力
2. 学会处理文本和语料
3. 学会典型 NLP 任务
4. 进入语音、大模型与 RAG

## 课程核心模块

- 基础层：Python 基础代码、文件与语料管理、Pandas 文本处理
- 方法层：文本清洗、规则抽取、spaCy、词频统计
- 应用层：文本分类、情感分析、主题与摘要、ASR 与 TTS
- 前沿层：大模型文本处理、RAG

## 课程强调什么

- 少讲空泛定义，多做可运行任务
- 每个工作台都指向明确输出
- 每个模板都服务作业、实验或项目
- 每个知识卡都回连工作台和输出动作
