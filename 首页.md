《Python 与自然语言处理》是一门以实践为主的课程，目标是让学生使用 Python 解决真实文本处理问题。

## 核心学习路径

1. Python 基础代码
2. 文件与语料管理
3. Pandas 文本数据处理
4. 文本清洗与规则抽取
5. spaCy 与词频分析
6. 分类、情感分析、主题与摘要
7. ASR / TTS、大模型、RAG
