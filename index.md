# Python 与自然语言处理

用 Python 处理真实文本问题，而不是只背术语。

这门课程从环境搭建、基础代码、文件与语料处理开始，逐步进入文本清洗、规则抽取、spaCy、文本分类、情感分析、摘要、大模型和 RAG。
